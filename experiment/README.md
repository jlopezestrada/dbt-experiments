Welcome to your new dbt project!

### Local Setup

To get started with this project locally, you'll need to set up your dbt profile:

1. Copy the example profile:
   ```bash
   cp profiles.yml.example profiles.yml
   ```
2. The default profile is configured to use DuckDB. You can modify `profiles.yml` if you need to change the database path or other settings.

### Using the starter project

Try running the following commands:
- dbt run
- dbt test
